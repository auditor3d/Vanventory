Connect Google Sheets

Your inventory will be saved to a Google Sheet you own — free, permanent, and accessible from any device.

  1. Go to script.google.com
  2. Create a new project
  3. Paste the Apps Script code (see instructions)
  4. Click Deploy → New deployment
  5. Choose Web app, set access to Anyone
  6. Copy the Web App URL and paste it below
